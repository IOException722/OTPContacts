Objective: To send OTP to mobile using Nexmo API
Functionality:
1) Contains two tablayout with view pager functionality
2) First tab contains contact list which implemented using RecyclerView and second tab is sent list
3) It use Volley library to get data from Nexmo API  - it return status-0 "Success" for register test number on Nexmo Dashboard (e.g Joey Number in contact list)

Libraries: Voley, RecyclerView
Rest API: Nexmo API
